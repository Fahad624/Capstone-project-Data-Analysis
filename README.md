# Data Analytics Capstone Project
## Overview

This project demonstrates the complete data analytics workflow using Excel, Looker Studio, Power BI, and SQL. The goal was to analyze Pakistan’s Largest E-Commerce Dataset and generate meaningful insights through various visualization and analytical tools.

## Project Components
#### Part 1 – Excel (Data Cleaning & Analysis)

- Used Power Query for data transformation and filtering.

- Created pivot tables, charts (histogram & pie chart), and used conditional formatting.

- Applied formulas to summarize and analyze product categories and payment methods.

#### Part 2 – Looker Studio (Dashboard Creation)

- Designed a multi-page interactive dashboard.

- Used slicers and date filters to explore data dynamically.

- Added 4–5 charts to visualize sales trends and performance metrics.

#### Part 3 – Power BI (Business Reporting)

- Built a 4-page Power BI report covering Men Fashion, Beauty & Grooming, and Kids & Baby categories.

- Added calculated columns and DAX measures (Total Sales, Products Sold, Total Customers).

- Created an Overview page summarizing all categories using interactive visuals and slicers.

#### Part 4 – SQL (Office Management System)

- Designed and implemented relational database tables: Departments and Employees.

- Performed joins and aggregate queries to retrieve department-wise salary summaries and manager relationships.
